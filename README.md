# voiceAgent

`voiceAgent` is a public-facing AI voice agent project focused on premium,
natural, low-latency spoken interaction.

## Project Documents

This project keeps these top-level files as its shared documentation contract:

- `README.md`: what the project is
- `AGENTS.md`: how coding agents should work in this repo
- `ARCHITECTURE.md`: how the system is structured
- `DESIGN.md`: how the UI should look and feel
- `DEVELOPMENT.md`: how to run, test, and build locally
- `MVP_PLAN.md`: product and execution plan
- `update_logs/`: commit-time development history

If you are building or modifying UI in this project, read `AGENTS.md` and
`DESIGN.md` first.

If you are starting a new Codex session, read `AGENTS.md`,
`ARCHITECTURE.md`, `DEVELOPMENT.md`, and the latest files in `update_logs/`
before changing code.

## Product Direction

The product should feel:

- premium and modern
- voice-native rather than generic SaaS
- technically credible and enterprise-ready
- visually bold without becoming noisy

## Design Direction

The visual system is inspired by the public ElevenLabs website and adapted for
this project:

- dark navy foundations
- pink, violet, and cyan accent lighting
- layered glass-like surfaces
- cinematic landing-page composition
- waveform, transcript, and voice-agent UI metaphors

See `./DESIGN.md` for the full design system.
